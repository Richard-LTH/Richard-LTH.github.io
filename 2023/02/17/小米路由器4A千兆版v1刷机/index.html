小米路由器4A千兆版v1刷机

小米路由器4A千兆版v1刷机

一.打开r4a的telnet功能

区分r4a版本，固件版本28开头为v1，30开头为v2

安装python后安装requests

安装命令：pip install requests

进入下载好的工具包（修改版，原版卡端口23报错），在文件目录下打开终端运行remote_command_execution_vulnerability.py文件，stock值在路由器后台网页url中。

二.备份编程器固件并刷入breed

完成后用telnet 192.168.31.1或使用putty连接路由器后台。用户名：root 密码：root

进入后台后备份路由器原厂固件

命令：cat /proc/mtd&&dd if=/dev/mtd0 of=/tmp/ALL_backup.bin

然后哦用winscp软件讲路由器的编程器固件下载保存到电脑。同时讲breed.bin上传至路由器。

三.刷机

备份eeprom.bin（信号相关数据备份）

选择网上下载的各版本（对应路由器）刷机包刷入固件，第一次刷入时勾选刷入eeprom保证信号稳定。

刷入完成后重启路由器进入路由器后台。
